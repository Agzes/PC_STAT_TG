Функции бота:
* В приложении:
  
  `1. Главная - небольшая информация, разные ссылки.`
  
  `2. Терминал - позволяет видеть действия которые происходят.`
  
  `3. Статистика - позволяет узнать загруженность: Цп, Гп, Озу. Также позволяет увидеть загруженность каждого ядра Цп по отдельности. С плагином имеет подробный вид. Имеет функцию авто-обновления.`
  
  `4. Плагины - включение/отключение плагинов (модулей) бота для разных функций.`
  
  `5. Лог - более продвинутая версия терминала`
  
  `6. Настройки - для изменения токена id и прочего`
  
* В боте:

  * Управление ботом:
    
    `1. Выключение - выкл. бота`
    
    `2. Информация - информация о создателе`

  * Управление музыкой/видео/звуком:
    
    `Звук - громче/тише/выкл`

    `Видео - ⬅/➡/пауза`

    `Музыка - след./прев./пауза`

  * Управление ПК:
 
    `1. Блокировка - блокирует пк (win + l)`

    `2. Выключение - выключает пк`
    
    `3. Перезагрузка - перезапускает пк`
    
    `4. Спящий режим - переводит пк в режим гибернации`
    
    `5. Информация о ПК - присылает информацию о Цп, Гп, ОЗУ и тд`
    
    `6. Скриншот - делает снимок экрана и присылает`

    `7. Enter - нажимает Enter`

    `8. Закрытие окна - закрывает окно (alt + f4)`

    `9. Свернуть всё - сворачивает все окна (win + d)`

    `10. Проводник - открывает проводник (win + e)`

    `11. Батарея - проверяет наличие батареи, проверяет подключение зарядки, проверяет заряд и высылает результат`

    `12. Корзина (очистка) - очищает корзину (возможен баг из-за недостатка прав)`

    `13. Список программ - показывает список всех запущенных приложений `
